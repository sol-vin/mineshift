# Mineshift

An interactive art piece. Explore an endless chasm up and down. See bridges, beams, and other features

## Usage

1. Install [raylib](https://github.com/sol-vin/raylib-cr#installation)
2. `shards install`
3. `crystal run ./src/mineshift`
